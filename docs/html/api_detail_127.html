1.接口说明

        设置用户扩展信息。

2.接口URL

        http://api.cloud.189.cn/updateUserInfoExt.action

3.支持格式

        xml

4.HTTP请求方式

        POST

5.请求Headers


字段标识 说明 数据类型 长度(字节) 可空
AccessToken OAuth2 Access Token String N
Signature 请求签名 String N
Date 请求日期和时间。例如：Wed, 01 Mar 2009 12:00:00 GMT String N

6.请求参数


字段标识 说明 数据类型 长度(字节) 可空
userExt 用户扩展信息元素容器 Container N
headPortrait multipart/form-data 要上传的头像图片，仅支持JPEG格式，图片尺寸必须为140*140，图片大小小于128KB。（建议使用上传用户头像接口） N
nickname String 昵称,为空时不更新 Y
gender String 性别(M-男  F-女),为空时不更新 Y
birthday String 生日(YYYYMMDD) ,为空时不更新 Y

7.请求样例 

无

8.返回参数


字段标识 说明 数据类型 长度(字节) 可空
userExt Container 用户扩展信息元素容器 N
—loginName String 用户登录名
—headPortraitUrl String 头像URL（原始尺寸）
—nickname String 昵称
—gender String 性别
—birthday String 生日(YYYYMMDD)
—realname String 真实姓名
—mobile String 手机
—email String 邮箱
—idNumber String 身份证号
—safeMobile String 安全手机
