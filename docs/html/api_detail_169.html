1.接口说明

        删除文件夹。

2.接口URL

        http://api.cloud.189.cn/deleteFolder.action

3.支持格式

        xml

4.HTTP请求方式

        POST

5.请求Headers


字段标识 说明 数据类型 长度(字节) 可空
AccessToken OAuth2 Access Token String N
Signature 请求签名 String N
Date 请求日期和时间。例如：Wed, 01 Mar 2009 12:00:00 GMT String N

6.请求参数


字段标识 说明 数据类型 长度(字节) 可空
folderId 要删除的文件夹Id Long N
forcedDelete 0或空 - 普通删除，放入回收站 Short Y

7.请求样例 

POST /deleteFolder.action HTTP/1.1

Host: api.cloud.189.cn

Date: date

AccessToken: key

Signature: signature



folderId=1001
