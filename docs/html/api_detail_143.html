1.接口说明

        获取指定文件的下载地址。

2.接口URL

        http://api.cloud.189.cn/getFileDownloadUrl.action

3.支持格式

        xml

4.HTTP请求方式

        GET

5.请求Headers


字段标识 说明 数据类型 长度(字节) 可空
AccessToken OAuth2 Access Token String N
Signature 请求签名 String N
Date 请求日期和时间。例如：Wed, 01 Mar 2009 12:00:00 GMT String N

6.请求参数


字段标识 说明 数据类型 长度(字节) 可空
fileId 文件ID Long N
short 是否获取短地址， true：获取短地址格式下载URL 为空或false：获取原始下载URL Boolean Y
forcedGet 1 - 强制获取下载地址（需配置应用权限） 0或空 -普通获取 Short Y

7.请求样例 

GET /getFileDownloadUrl.action?fileId=81 HTTP/1.1

Host: api.cloud.189.cn

Date: date

AccessToken: key

Signature: signature

8.返回参数


字段标识 说明 数据类型 长度(字节) 可空
fileDownloadUrl 文件下载地址 String N

9.返回样例 

正确返回样例：			url


错误返回样例：无
