1.接口说明

        PUT方式上传文件数据并创建文件，不支持断点续传。

2.接口URL

        http://upload.cloud.189.cn/uploadFile.action

3.支持格式

        xml

4.HTTP请求方式

        PUT

5.请求Headers


字段标识 说明 数据类型 长度(字节) 可空
AccessToken OAuth2 Access Token String N
Signature 请求签名 String N
Date 请求日期和时间。例如：Wed, 01 Mar 2009 12:00:00 GMT String N

6.请求参数


字段标识 说明 数据类型 长度(字节) 可空

7.请求样例 

PUT /putFile.action HTTP/1.1

Host: api.cloud.189.cn

Date: date

AccessToken: key

Signature: signature

Edrive-ParentFolderId: 1234

Edrive-FileName: test.txt

Edrive-FileMD5: 9999999999999999999999999999999

Content-Length: 1024



[二进制数据]

8.返回参数


字段标识 说明 数据类型 长度(字节) 可空
file 文件元素容器 Container N
—id 文件Id Long
—name 文件名称 String
—size 文件大小 Long
—md5 文件的MD5码 String
—createDate 文件夹创建日期(YYYY-MM-DD hh:mm:ss) String
—rev 文件夹版本号 Long

9.返回样例 

正确返回样例：1999test.txt102499999999999999999999999999999992011-03-24 09:00:0020110324090000


错误返回样例：无
