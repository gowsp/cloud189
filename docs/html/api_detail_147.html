1.接口说明

        在指定的文件夹中新建文件夹。

2.接口URL

        http://api.cloud.189.cn/createFolder.action

3.支持格式

        xml

4.HTTP请求方式

        POST

5.请求Headers


字段标识 说明 数据类型 长度(字节) 可空
AccessToken OAuth2 Access Token String N
Signature 请求签名 String N
Date 请求日期和时间。例如：Wed, 01 Mar 2009 12:00:00 GMT String N

6.请求参数


字段标识 说明 数据类型 长度(字节) 可空
parentFolderId 父文件夹ID，为空时，新建文件夹父目录为应用默认文件夹 Long Y
relativePath 相对于parentFolderId的目录路径，以 / 分隔，支持创建多级目录，无论相对路径中的目录是否存在 String Y
folderName 新建文件夹名称 String N

7.请求样例 

POST /getFileDownloadUrl.action?fileId=81 HTTP/1.1

Host: api.cloud.189.cn

Date: date

AccessToken: key

Signature: signature



parentFolderId=1234&folderName=doc

8.返回参数


字段标识 说明 数据类型 长度(字节) 可空
folder 新建文件夹元素容器 Containter N
—id 文件夹Id Long
—name 文件夹名称 String
—createDate 文件夹创建日期(YYYY-MM-DD hh:mm:ss) String
—rev 文件夹版本号 Long

9.返回样例 

正确返回样例：81doc2009-02-06 12:00:0020110811120000


错误返回样例：无
